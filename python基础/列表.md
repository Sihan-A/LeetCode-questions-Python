    - 二维数组，根据第一项大小排序

        ```python
        list_2d = [[20,30],
                   [10,30],
                   [15,20],
                   [5,10],]
        list_2d.sort(key=lambda x:x[0])
        print(list_2d) # [[5, 10], [10, 30], [15, 20], [20, 30]]
        ```

    ```python
    m, n = 2, 3
    list_2d = [[0 for _ in range(n)] for _ in range(m)]
    ```
